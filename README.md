🏭 Production Line Efficiency & Downtime Analysis Dashboard
1. Project Title / Headline
🏭 Production Line Efficiency & Downtime Analysis Dashboard

An interactive Power BI dashboard designed to monitor production line performance by analyzing batch processing time, operator efficiency, machine downtime, and production trends. The dashboard helps manufacturing teams identify operational bottlenecks, reduce downtime, and improve production efficiency through data-driven insights.

2. Short Description / Purpose
The Production Line Efficiency & Downtime Analysis Dashboard provides a centralized view of manufacturing operations across different beverage production lines. It enables production managers, plant supervisors, and operations teams to monitor batch performance, operator efficiency, downtime causes, and monthly production trends, supporting faster operational decisions and continuous process improvement.

3. Tech Stack
The dashboard was built using the following tools and technologies:
📊 Power BI Desktop – Used for designing and developing the interactive dashboard.
📂 Power Query – Used for cleaning, transforming, and preparing manufacturing data.
🧠 DAX (Data Analysis Expressions) – Used to create calculated measures, KPIs, and dynamic aggregations.
🔗 Data Modeling – Established relationships between production, operators, downtime, and product datasets.
📈 Interactive Visualizations – KPI Cards, Bar Charts, Line Charts, Pie Chart, Treemap, Funnel Chart, and Slicers.
📁 File Format – .pbix for development and .png for dashboard previews.
4. Data Source

Source: Simulated Manufacturing Production Dataset from Kaggle
The dataset represents manufacturing operations from a beverage production facility and contains information related to production batches, processing time, operators, machine downtime, products, and operational errors.
The dataset includes:
Batch IDs
Production Dates
Product Types
Batch Processing Time
Operator Details
Operator Error Status
Downtime Duration
Downtime Factors
Production Line Information
Monthly Production Records
The data was cleaned and transformed using Power Query before visualization.

5. Features / Highlights
• Business Problem
Manufacturing plants process numerous production batches daily, making it difficult to manually monitor operational efficiency. Production managers need quick visibility into processing time, downtime causes, and operator performance to minimize delays and improve productivity.

Some common business questions include:
Which products require the longest batch processing time?
Which downtime factors affect production the most?
Which operators are associated with production delays?
How does production time change over different months?
Which products experience the highest downtime?
Are operator errors impacting production efficiency?

Without a centralized dashboard, answering these questions requires time-consuming manual analysis.

• Goal of the Dashboard
The dashboard was designed to:
Monitor overall production performance.
Analyze batch processing time across products.
Track machine downtime and downtime causes.
Evaluate operator performance and error rates.
Identify products contributing to higher downtime.
Support operational decision-making through interactive visual analytics.
• Walkthrough of Key Visuals
Executive KPI Cards

Provides an overview of the production process using key metrics:
Total Batches Processed: 38
Total Batch Time: 3,858 minutes
Total Products: 6
Total Operators: 4
Operator Error Rate: 0.56%

These KPIs offer a quick summary of overall production performance.
Batch Time by Product (Horizontal Bar Chart)
Compares the total batch processing time across different beverage products such as CO-600, CO-2L, RB-600, LE-600, DC-600, and OR-600.
This visualization helps identify products requiring longer production cycles and highlights opportunities for process optimization.
Minimum Batch Time by Product (Line Chart)
Displays the minimum batch time achieved for each product, enabling managers to compare operational efficiency across production lines and identify best-case manufacturing performance.
Operator vs Time Taken (Pie Chart)
Shows the distribution of total batch processing time among operators.
This visual helps evaluate workload allocation and compare operator productivity.
Downtime by Factor (Funnel Chart)
Ranks downtime causes based on their impact on production.

Typical factors include:
Machine Failure
Inventory Shortage
Batch Change
Machine Adjustment
Other Operational Interruptions

This visualization enables maintenance teams to prioritize recurring production issues.
Product Downtime Analysis (Treemap)
Displays downtime distribution across different products and associated downtime reasons.
The treemap quickly highlights which products experience the greatest production interruptions and the primary causes behind them.
Batch Time by Month (Area Line Chart)
Tracks production batch time over different months.
This visual supports trend analysis by identifying increases or decreases in production time and helping managers monitor operational improvements over time.

Interactive Filters
Users can dynamically filter the dashboard using:
Date
Operator Error Status
Product

All visualizations update automatically based on selected filters, enabling detailed analysis from multiple operational perspectives.
• Business Impact & Insights
Production Optimization
Identifies products with higher processing times, helping managers optimize production scheduling.
Downtime Reduction
Highlights the most significant downtime factors, allowing maintenance teams to focus on the biggest operational challenges.
Workforce Performance Monitoring
Provides visibility into operator workload and error rates, supporting better workforce planning and targeted training initiatives.
Improved Decision Making
Interactive filtering enables managers to investigate production performance by product, operator, and time period, leading to faster and more informed decisions.
Enhanced Operational Efficiency
By consolidating production, downtime, and operator data into a single dashboard, stakeholders can quickly identify inefficiencies, reduce delays, and improve overall manufacturing performance.
